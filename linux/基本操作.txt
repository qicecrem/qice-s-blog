打开命令行界面（终端）：ctrl+alt+t
ip link：显示网络接口的链接状态
ip addr：显示网络接口的详细信息
cd .. 返回上级目录
cd / 返回根目录
cd ~ 返回用户目录
安装软件：sudo apt-get install 软件名
安装deb包：sudo dpkg -i 软件名.deb
pwd：显示当前所在目录。
mkdir 文件夹名：创建文件夹
mkdir -p 文件夹名/文件夹名：自动创建不存在的父目录。
touch 文件名：创建文件
cat 文件名：查看文件内容
more 文件名：分页查看文件内容(空格翻页，q退出)
cp [-r] 源文件 目标文件  ：复制文件,-r可选用于复制文件夹。
mv 源文件 目标文件  ：移动文件或重命名文件。
rm [-r -f] 文件名 ：删除文件，-r可选用于删除文件夹,-f可选用于强制删除。
* : 通配符，匹配任意字符包含空格。
which 命令名：查看命令所在位置。
find 路径 -name 文件名：以文件名查找文件。
find 路径 -size +|-n[kMG]：以文件大小查找文件,+表示大于，-表示小于，n表示大小，k表示KB，M表示MB，G表示GB。
grep [-n] 关键字 文件路径 ：从文件中通过关键字过滤内容，-n可选用于显示行号。
wc [-c -m -l -w] 文件路径 ：统计文件内容，-c可选用于统计字节数，-m可选用于统计字符数，-l可选用于统计行数，-w可选用于统计单词数。
|:管道符，将前一个命令的输出作为后一个命令的输入。
echo 输出内容：输出内容到控制台。
echo 输出内容 > 文件路径：将输出内容写入文件。
echo 输出内容 >> 文件路径：将输出内容追加到文件。
·：反引号，将反引号内的命令执行结果作为命令。
tail [-f -num] 文件路径：查看文件内容，-f可选用于实时查看文件内容，-num可选用于查看最后几行。


vim 文件名： 打开文件或创建新文件
进入插入模式：在正常模式下按 i 或 a 。
返回正常模式：在插入模式下按 Esc 。
进入命令模式：在正常模式下按 : 。
进入可视模式：在正常模式下按 v 。
正常模式
移动光标：
h：左移
j：下移
k：上移
l：右移
复制、剪切、粘贴：
yy：复制当前行
dd：剪切当前行
p：在光标后粘贴
P：在光标前粘贴
撤销、重做：
u：撤销
Ctrl + r：重做
删除：
x：删除光标所在字符
dw：删除到单词结尾
d$：删除到行尾

插入模式
i：在光标前插入
a：在光标后插入
o：在当前行下方插入新行
O：在当前行上方插入新行
命令模式
保存文件：
:w：保存文件
:w filename：另存为文件
退出 Vim：
:q：退出
:q!：强制退出不保存
:wq 或 :x：保存并退出

可视模式
选择文本：
v：进入可视模式，选择字符
V：选择整行
Ctrl + v：选择矩形块
操作选择：
y：复制选择部分
d：剪切选择部分
p：粘贴
搜索与替换
搜索：
/pattern：向下搜索
?pattern：向上搜索
n：下一个匹配
N：上一个匹配
替换：
:s/old/new/g：在当前行替换所有匹配
:%s/old/new/g：在整个文件替换所有匹配
:%s/old/new/gc：在整个文件替换所有匹配，并确认

su [-] [用户名]： 切换用户,-表示切换用户后加载环境变量，不指定用户名则切换到root用户，用exit命令退回上一个用户，也可用快捷键Ctrl+d。

sudo认证：切换到root用户，执行visudo命令，，会自动通过vi编辑器打开：/etc/sudoers，
在文件最后添加：用户名 ALL=（ALL） NOPASSWD:ALL 保存退出。

groupadd 组名：创建新组
groupdel 组名：删除组
useradd [-g -d] 用户名：创建新用户，-g指定组，-d指定用户目录.不指定-g会创建同名组并自动加入，不指定-d则默认创建/home/用户名目录。
userdel [-r] 用户名：删除用户，-r可选用于删除用户目录。
id [-u -g -n] 用户名：显示用户信息，-u显示用户ID，-g显示用户组ID，-n显示用户名称。
passwd 用户名：修改用户密码。
su 用户名：切换用户。
usermod -aG 组 用户名：将用户加入组。
getent passwd：显示所有用户及其信息。
getent group：显示所有组及其信息。

文件权限：-表示文件，d表示文件夹，l表示链接文件，r表示可读，w表示可写，x表示可执行。
chmod [-R] 权限 文件或文件夹：修改文件权限，-R可选用于递归修改文件夹及其内容。
r记为4，w记为2，x记为1。

chown [-R] [用户名] [:] [用户组] 文件或文件夹。-R可选用于递归修改文件夹及其内容，用户可选用于修改所属用户，用户组可选用于修改所属用户组，:可选用于分隔用户和用户组。

ctrl+c：强制停止。
history：显示历史命令。
!命令前缀，自动执行上一次匹配前缀的命令。
清屏：clear，ctrl+l。

yum [-y] [install] [remove] [search] 软件名称：安装、卸载、搜索软件。-y可选用于自动回答yes。Centos
apt-get [-y] [install] [remove] [search] 软件名称：安装、卸载、搜索软件。-y可选用于自动回答yes。Ubuntu/Debian

systemctl start|stop|status|enable|disable 服务名：启动、停止、查看服务状态、开启开机自启动、关闭开机自启动。

ln [-s] 源文件 目标文件：创建链接文件，-s可选用于创建软链接。

data [-d] [+格式化字符串] -d 按照给定的字符串显示日期，一般用于日期计算。
格式化字符串：通过特定的字符串标记，来控制显示的日期格式，%Y表示年，%m表示月，%d表示日，%H表示时，%M表示分，%S表示秒。

特殊的ip地址：
127.0.0.1：本机回环地址，用于本机测试。
0.0.0.0：表示任何地址，用于服务器上绑定所有IP地址;代指本机；端口绑定中用来确定绑定关系。

hostname：显示当前主机名。
hostnameectl set-hostname 主机名：修改主机名。

ping [-c num ] ip或主机名：测试网络连通性，-c num可选用于指定测试次数。

wget [-b] url ：下载文件，-b可选用于后台下载。
curl [-O 文件名] url：下载文件，-O可选用于指定下载文件名。

端口：
公认端口：0~1023，由IANA管理，一般用于知名服务。
注册端口：1024~49151，由IANA管理，一般用于私有服务。
动态/私有端口：49152~65535，由操作系统自动分配，一般用于临时服务。
namp IP地址：查看指定IP地址的端口使用情况。

ps [-e -f] : 显示进程信息，-e显示所有进程，-f显示完整信息。

kill [-9] 进程ID：杀死进程，-9可选用于强制杀死。

top：显示系统进程信息。
df [-h] : 显示磁盘使用情况，-h可选用于显示更友好的容量单位。
iostat [-d -x -k -m -p -t] [设备名]：显示磁盘IO信息，-d显示磁盘IO，-x显示详细信息，-k显示以KB为单位，-m显示以MB为单位，-p显示指定设备，-t显示间隔时间。
sar -n DEV [interval]：显示网络IO信息，DEV表示网卡设备名，interval表示间隔时间。

环境变量：
PATH：搜索命令的路径。
通过$符可以查看环境变量的值。
临时生效：
export 变量名=值：设置环境变量。
source 脚本名：执行脚本。
永久生效：
对用户 ~/.bashrc 文件添加环境变量。
对所有用户 /etc/profile 文件添加环境变量。

加入自定义的命令搜索路径：
PATH=$PATH:自定义路径：在PATH变量后面加上自定义路径。
export PATH=$PATH:自定义路径：临时生效。

tar [-c -v -x -f -z -C] 参数1 参数2...参数n.-c创建，-v显示过程，-x解压，-f指定文件名，-z压缩，-C指定解压目录。

压缩：
tar -cvf 压缩包名.tar 源文件：创建tar包，简单封装。
tar -zcvf 压缩包名.tar.gz 源文件：创建tar.gz包,使用gzip模式，体积压缩效果最好。
解压：
tar -xvf 压缩包名.tar：解压tar包到当前目录。
tar -xvf 压缩包名.tar -C 解压目录：解压tar包到指定目录。
tar -zxvf 压缩包名.tar.gz：解压tar.gz包到当前目录。

zip [-r] 压缩包名 源文件：创建zip包，若有文件夹需要使用-r可选用于复制文件夹。
unzip 压缩包名.zip：解压zip包到当前目录。
unzip -d 解压目录 压缩包名.zip：解压zip包到指定目录。

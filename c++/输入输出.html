8.cout.setf(格式控制符)：用于设置输出格式
常用的格式标志符：
std::ios::boolalpha: 将布尔值显示为 true 或 false。
std::ios::showbase: 显示整数值的基数前缀（例如十六进制的 0x，八进制的 0）。
std::ios::showpoint: 始终显示浮点数的小数点。
std::ios::showpos: 对正数显示 + 号。
std::ios::uppercase: 使用大写字母表示十六进制数字和科学记数法。
std::ios::hex: 将数值设为十六进制格式。
std::ios::oct: 将数值设为八进制格式。
std::ios::dec: 将数值设为十进制格式。
std::ios::fixed: 使用定点记数法表示浮点数。
std::ios::scientific: 使用科学记数法表示浮点数。
std::cout.setf(std::ios::left, std::ios::adjustfield);左对齐
实例：


        #include 

            int main() {
                int number = 255;
            
                // 设置十六进制格式、基数前缀和大写字母
                std::cout.setf(std::ios::hex, std::ios::basefield);
                std::cout.setf(std::ios::showbase);
                std::cout.setf(std::ios::uppercase);
            
                std::cout << "十六进制: " << number << std::endl;  // 输出: 0XFF
            
                // 取消大写字母标志
                std::cout.unsetf(std::ios::uppercase);
            
                std::cout << "十六进制（无大写）: " << number << std::endl;  // 输出: 0xff
            
                // 重置为十进制格式
                std::cout.setf(std::ios::dec, std::ios::basefield);
            
                std::cout << "十进制: " << number << std::endl;  // 输出: 255
            
                return 0;
            }

9.cout.unsetf(格式控制符)：用于取消输出格式
10.输入输出流的格式控制：
(1).设置精度：cout.precision(n)：用于设置浮点数的精度，n为精度值

        #include 
        int main() {
            std::cout.precision(3);
            std::cout << 123.456789 << std::endl;
            return 0;
        }
(2).设置宽度：cout.width(n)：用于设置输出宽度，n为宽度值

        #include 

            int main() {
                std::cout.width(10);
                std::cout << 123 << std::endl;
                return 0;
            }
(3).设置填充字符：cout.fill(字符)：用于设置输出填充字符

        #include 

            int main() {
                std::cout.fill('*');
                std::cout.width(10);
                std::cout << 123 << std::endl;
                return 0;
            }
